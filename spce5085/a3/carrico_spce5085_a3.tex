\problem{1}{Calculate the system noise temperature of a 2 GHz receiver, given the following: $T_{in} = 20 K, T_{RF} = 45, T_{m} = 450 K, T_{IF} = 800 K, G_{m} = 1 dB, G_{RF} = 20 dB, G_{IF} = 30 dB$.
See text section 4.3.2 (15 pts); Where $T_{in}$ = Temperature In, $T_{RF}$ = Temperature of RF elements, $T_{m}$ = Temperature of mixer, $T_{IF}$ = Temperature of Intermediate Frequency, and $G_{m}$ = gain of mixer, $G_{IF}$ = gain of IF.
Reminder: Use consistent units: don't mix dB and linear, convert dB units to linear before using equation 4.20.}
\solution{
Convert from dB to linear:
$$G_{m} = 10^{1/10} = 1.259$$
$$G_{RF} = 10^{20/10} = 100$$
$$G_{IF} = 10^{30/10} = 1000$$

System noise temperature:
$$T_{S} = T_{RF} + T_{in} + \frac{T_{m}}{G_{RF}} + \frac{T_{IF}}{G_{m}G_{RF}}$$
$$T_{S} = 45 + 20 + \frac{450}{100} + \frac{800}{(1.259)(100)}$$
$$\ans{T_{S} \approx 75.85 \text{ K}}$$
}

\problem{4}{Using 14 GHz: A Ku-band satellite uplink has a carrier frequency of 14.0 GHz and a carries a symbol stream at $R_{s} = 20 Msps$. The transmitter and receiver have SRRC filters with $\alpha = .20$}
\subproblem{a}{What is bandwidth occupied by the RF signal?}
\solution{
With matched SRRC filters at transmit and receive, the end-to-end response is a raised-cosine filter, so the occupied RF bandwidth is
$$B = R_{s}(1+\alpha)$$
$$B = 20\text{ Msps} \times (1 + 0.20)$$
$$\ans{B = 24 \text{ MHz}}$$
}
\subproblem{b}{What is the frequency range of the transmitted RF signal?}
\solution{
The signal spans $\pm B/2$ about the carrier:
$$f_{c} - \frac{B}{2} \leq f_{RF} \leq f_{c} + \frac{B}{2}$$
$$14.000\text{ GHz} - 12\text{ MHz} \leq f_{RF} \leq 14.000\text{ GHz} + 12\text{ MHz}$$
$$\ans{13.988 \text{ GHz} \leq f_{RF} \leq 14.012 \text{ GHz}}$$
}
